解读

首先，调用 createUpdateEffect 函数，传入 useEffect 函数。createUpdateEffect 函数接收一个函数作为参数，返回一个新函数。这样封装的目的是为了支持 useLayoutEffect 的用法。ahooks 中有很多类似的这种封装，因为需要同时支持 useEffect 和 useLayoutEffect 的用法。
